# Yander — Holding Page

The launch holding page for **Yander**, a scenic-route planner for drivers, cyclists, and walkers.

**Live site:** [yander.app](https://yander.app)

## Stack

Pure static site — three files, zero dependencies.

- `index.html` — markup
- `styles.css` — design tokens, layout, motion
- `assets/yander-emblem.png` — logo

## Deploy

Hosted on [Cloudflare Pages](https://pages.cloudflare.com/), auto-deploying from `main`.
